# 12Է դասարան • Վերջին Զանգի հրավիրատոմս

Սա ամբողջովին ստատիկ կայք է, որը աշխատում է GitHub Pages-ում առանց backend-ի։

## Ինչ կա նախագծում

- `index.html` - հիմնական էջ, որը բացվում է որպես նամակ ուսուցչին
- `styles.css` - դիզայն + անիմացիաներ
- `app.js` - հետհաշվարկ, անձնականացում, քարտերի գեներացում
- `data/students-db.js` - աշակերտների տվյալներ
- `data/teachers-db.js` - ուսուցիչների տվյալների բազա
- `IMG/` - բոլոր լուսանկարները

## Ժամաչափ

Ժամաչափը հաշվարկում է մինչև՝ **21.05.2026, ժամը 10:00 (UTC+4 / Yerevan)**։

`app.js` ֆայլում այս տողն է վերահսկում օրը/ժամը.

```js
const EVENT_DATE = new Date("2026-05-21T10:00:00+04:00");
```

## Անձնականացված հրավեր ուսուցիչների համար

Յուրաքանչյուր ուսուցիչի համար տվեք առանձին հղում.

```txt
https://YOUR-DOMAIN/index.html?teacher=matematika
```

`teacher`-ի արժեքը վերցվում է `data/teachers-db.js`-ի `id` դաշտից։

Օրինակ՝

```js
{
  id: "matematika",
  name: "Մաթեմատիկայի ուսուցիչ",
  role: "Մաթեմատիկա",
  personalWish: "...",
  highlightStudents: ["Նարեկ", "Արմեն Ամիրբարյան"],
  heroImage: "3 guys, crazy photo.JPG",
  theme: "sky"
}
```

## Արագ նամակ անունով (առանց DB)

Եթե պետք է արագ ուղարկել մեկ հղում կոնկրետ անունով.

```txt
https://YOUR-DOMAIN/index.html?name=Աննա
```

Այս դեպքում էջը կսկսվի՝ `Սիրելի՛ Աննա` ձևով։

## Նոր ուսուցիչ ավելացնելու քայլեր

1. Բացեք `data/teachers-db.js`
2. Ավելացրեք նոր օբյեկտ `id` դաշտով
3. Պահեք ֆայլը
4. Բացեք կայքը և պատճենեք պատրաստի հղումը
